# VNOS project

This repository contains my project for the VNOS (embedded systems) course at FIIT STU.

## Used components

* Raspberry Pi 3B+
* 2x DS18B20 thermometer
* BH1750FVI light sensor
* 4-digit 7-segment display with 2x 74HC565 shift registers
* RGB LED module
* A button module
* A breadboard
* 4.7kOhm resistor
